# Welcome to Mecenate 👋

This is an [Expo](https://expo.dev) project created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```

3. Start to android

   ```bash
   expo start --android
   ```
   
4. Start to ios

   ```bash
   expo start --ios
   ```
